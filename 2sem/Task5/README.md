# Yacc Syntax Analyzers

Лабораторная работа по изучению генератора синтаксических анализаторов Yacc (Bison). В проекте реализованы две программы для анализа скобочных систем и вычисления арифметических выражений на основе формальных грамматик.

## Состав проекта

### 1. Вычисление протяжения скобочной системы (extension.y)
Программа определяет протяжение скобочной последовательности. Согласно определению, протяжение — это максимальное число неделимых систем, сцепление которых дает данную систему:
* Пустая цепочка $\epsilon$ имеет протяжение 0.
* Непустая система вида (S)S имеет протяжение на 1 больше, чем для «хвостовой» подсистемы S.
* Пример: ()(()()()) имеет протяжение 2, а ((())) — протяжение 1.

### 2. Калькулятор выражений (calc.y)
Интерактивный калькулятор для вычисления значений целочисленных выражений.
* Поддерживаемые операции: Сложение (+), вычитание (-), умножение (*), деление (/).
* Особенности:
    * Учет приоритетов операций: умножение и деление выше сложения и вычитания.
    * Все операции левоассоциативны.
    * Поддержка круглых скобок для изменения порядка действий.
    * Автоматическое построение дерева вывода восходящим методом.
## Технологический стек
* Язык реализации: C++ (внутри семантических действий Yacc).
* Генератор парсеров: Bison (совместим с Yacc).
* Компилятор: G++.
