# Model SQL Interpreter (C++)

Учебный проект по разработке интерпретатора подмножества языка SQL. Выполнен в рамках практикума на факультете ВМК МГУ. 

Данная версия поддерживает не только проверку структуры (синтаксис), но и семантический анализ с сохранением состояния таблиц в памяти.

---

## 🌟 Новые возможности (v2.0)

*   Persistent Memory (Семантическая память): Интерпретатор запоминает созданные таблицы. Вы можете создать таблицу одной командой, а затем обратиться к ней другой — данные о структуре сохраняются
*   Semantic Validation: 
    *   Проверка существования таблиц перед выполнением SELECT или INSERT
    *   Проверка наличия конкретных столбцов в таблице
    *   Запрет на создание дубликатов таблиц
*   Alias Support: Поддержка синтаксиса SELECT column AS alias
*   Recursive Descent: Полноценный разбор сложных логических выражений в блоке WHERE (AND, OR, NOT, скобки).

---

## 🛠 Технологии

*   Язык: C++ (стандарт C++11 и выше).
*   Архитектура: 
    *   Scanner: Лексический анализатор (автоматный метод)
    *   Parser: Синтаксический и семантический анализатор (метод рекурсивного спуска)
