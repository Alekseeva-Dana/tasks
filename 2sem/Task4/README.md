# Simple SQL Interpreter

Учебный проект по реализации интерпретатора подмножества языка SQL на языке C++. Основная цель проекта — изучение принципов построения компиляторов, синтаксического анализа и реализации виртуальных машин для вычисления выражений.

## 🌟 Ключевые особенности

- Лексический анализ: Собственный сканер (Lexer) для разбиения запросов на токены.
- Синтаксический анализ: Рекурсивный нисходящий парсер (Recursive Descent Parser).
- ПОЛИЗ (RPN): Перевод сложных условий в блоке WHERE в обратную польскую запись для учета приоритетов операций.
- Execution Engine: Собственная стековая машина для выполнения условий и фильтрации данных в реальном времени.

## 🏗 Архитектура

Проект состоит из трех основных уровней:
1. Frontend: Лексический разбор (Scanner).
2. Middle-end: Синтаксическая проверка и генерация постфиксного кода (POLIZ).
3. Backend: Исполнение запроса (Exec) и работа с виртуальными таблицами.

### Логика обработки WHERE
Для корректной работы с приоритетами (NOT > AND > OR), условие WHERE разбирается в иерархическом порядке и преобразуется в стек команд.
Например, условие:
WHERE age > 18 AND status = 'active'
превращается в ПОЛИЗ:
age 18 > status 'active' = AND
