    <h1>Модельный SQL-интерпретатор</h1>
    <p>
        <span class="tag">C++</span>
        <span class="tag">SQL</span>
        <span class="tag">Interpreter</span>
        <span class="tag">MSU VMK</span>
    </p>
    <p>Учебный проект по разработке интерпретатора подмножества языка SQL. Выполнен в рамках практикума на факультете ВМК МГУ по методическому пособию Волкова И.А. и др.</p>
    <h2>🚀 Возможности</h2>
    <ul>
        <li><b>Лексический анализ:</b> разбиение запроса на токены (ключевые слова, идентификаторы, константы).</li>
        <li><b>Синтаксический анализ:</b> проверка структуры команд методом рекурсивного спуска.</li>
        <li><b>Интерактивный режим (REPL):</b> ввод команд напрямую в консоль.</li>
        <li><b>Логирование:</b> одновременный вывод результатов лексера и парсера для отладки.</li>
    </ul>
    <h2>🛠 Сборка и запуск</h2>
    <p>Для компиляции используется <code>Makefile</code>. В терминале выполните:</p>
    <pre>
make clean
make
./sql</pre>
    <h2>📋 Поддерживаемые команды</h2>
    <p>Интерпретатор понимает следующие конструкции:</p>
    <ul>
        <li><code>SELECT * FROM table WHERE condition;</code></li>
        <li><code>CREATE TABLE name (col1 type1, col2 type2);</code></li>
        <li><code>INSERT INTO table VALUES (val1, val2);</code></li>
    </ul>
    <h2>📂 Структура проекта</h2>
    <ul>
        <li><code>scanner.h / .cpp</code> — Лексический анализатор.</li>
        <li><code>parser.h / .cpp</code> — Синтаксический анализатор.</li>
        <li><code>main.cpp</code> — Точка входа и интерактивный интерфейс.</li>
        <li><code>Makefile</code> — Скрипт сборки.</li>
    </ul>
